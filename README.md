# schk
Le but de notre projet c’est de créer un site web et une application mobile.
Leur contenu serait une manière d’inciter le citoyen à recycler ses déchets d’une facon appropriée, en échange il peut gagner des superpoints qui lui serviront à les dépenser dans des commerces locaux (garantir le commerce local) tels que boucherie, magasin de vélos, épicerie, ou des grandes chaines comme Migros, Coop, Mcdonald’s
